# Django project: Разработка проекта по созданию интернет-магазина

## Произведена настройка проекта, создано приложение catalog со страницами home и contacts.

## Установка:
1. Клонируйте репозиторий по SHH-ключу:
    ```
    git@github.com:annvikky/DjangoProject_hw.git
    ```
2. Установите зависимости:
    ```
    pip install -r requirements.txt
    ```
